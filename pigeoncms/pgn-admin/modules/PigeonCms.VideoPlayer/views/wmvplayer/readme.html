Quickstart

The easiest way to get to know the player is by using the setup wizard. Select an example, set the file or playlist you want to play and copy-paste the embed code to your site.

Wiki

The JW Player Wiki contains a wealth of information about the player, including:

  2. All supported variables for customizing the player.
  3. The javascript API documentation.
  4. All previous versions of the player.
